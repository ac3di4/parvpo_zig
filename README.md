## Детали реализации
Папка для результатов работы `/home/students/zig`

Чтобы автоматически собрать и запустить `debug` версию, необходимо прописать `make`. \
Для просто сборки `make debug`

Если нужны исполняемые файлы `windows.exe` и `linux` как в задании - то `make release`

Исполняемые файлы искать в папке `bin/`\
Подробнее о режимах сборки в `Makefile`
